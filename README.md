🏥 Hospital Dashboard – Excel Analytics Project

An interactive Hospital Analytics Dashboard developed in Microsoft Excel to monitor hospital performance, patient demographics, revenue metrics, admission trends, and healthcare insights.

Features:

Total Patients KPI,
Revenue Analysis,
Average Billing Metrics,
Medical Condition Analysis,
Blood Group Distribution,
Gender & Age Analysis,
Admission Type Insights,
Hospital & Doctor Rankings,
Interactive Filters (Slicers).

Tools: Excel, Pivot Tables, Pivot Charts, Slicers, Data Visualization

Objective: Transform healthcare data into actionable insights through an easy-to-understand and interactive dashboard.
